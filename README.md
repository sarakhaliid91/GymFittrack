# GymFittrack 🏋️‍♀️

Following up on my fitness journey

تطبيق ويب لمتابعة تماريني في النادي وقياسات جسمي 💪

## الصفحات

- **🏋️ تماريني**: تسجيل أوزان تماريني حسب تقسيمة الجدول (علوي/سفلي 1، علوي/سفلي 2، علوي Pull، علوي Push)، مع اختيار نوع المعدة (آلة / دمبل / كيبل / بار / وزن الجسم) وتاريخ كل تمرين.
- **📏 قياساتي**: تسجيل قياسات الجسم (الوزن، الصدر، الخصر، اسفل البطن، محيط الارداف، الفخذ، البطات، الزند) مع التاريخ.
- **🔥 تطوري**: ملخص للاستمرارية (Streak)، عدد أيام التمرين، تطور الأوزان لكل تمرين، وتطور القياسات بمرور الوقت.

## التشغيل محليًا

```bash
npm install
npm run dev
```

البيانات تُحفظ محليًا في المتصفح (localStorage).
